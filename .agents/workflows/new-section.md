---
description: Auxiliar o usuário a criar seções no sistema
---

1. Pergunte ao usuário o nome da nova seção (ex: header, lista, galeria).
2. Crie uma estrutura html semântica ('<section>') com um ID correspondente.
3. adicione um comentário no css/js indicando onde começa a logica dessa nova parte.
4. aplique um padding padrão 'p-6' e bordas 'border-zinc-200'.
